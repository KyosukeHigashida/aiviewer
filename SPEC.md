# SPEC

## 目的
研究・技術文書の読解時に、PDF の該当箇所へ注釈と議論を残せるローカルファーストな PDF ビューアを作る。

## 主要ユースケース
- ローカルの PDF を開いて読む。
- PDF 上で選択したテキスト範囲に対してスレッドを作る。
- スレッドにコメントを追加し、あとで再表示する。
- 右側の常設コメント欄で注釈一覧と内容を確認する。
- 保存済み注釈を一覧または PDF 上のマーカーから再発見する。

## MVP 範囲
- React + TypeScript + Vite を前提に実装する。
- 画面は左約 75% を PDF 表示、右約 25% を常設コメント欄とする。
- デスクトップでは PDF とスレッドペインを独立スクロールできるようにする。
- モバイルでは下固定のスレッドペインを使い、閉状態でも引き出し操作ができるようにする。
- PDF 表示ライブラリは `react-pdf` を使用する。
- 対象は単一 PDF とする。
- 最初はファイル選択からローカル PDF を開く。
- 注釈対象は PDF 上で選択したテキスト範囲に限定する。
- 注釈スレッドとコメント、PDF 本体、前回文書の再開状態をブラウザ内に保存し、再起動後に復元できるようにする。
- 保存済み注釈は、右ペインの一覧、PDF 上の位置マーカー、本文ハイライトから再発見できるようにする。
- 新規保存分の本文ハイライトは、保存済み矩形を優先して再描画する。
- 永続化は `IndexedDB` を使用する。
- 単一ユーザー、同一端末の同一ブラウザ利用を前提とする。

## 非目標
- バックエンドの構築
- ユーザー認証
- クラウド同期
- 複数人コラボレーション
- 注釈の共有、公開、インポート、エクスポート
- 複数 PDF の高度な管理機能
- 画像領域への注釈
- OCR や AI による自動要約、自動分類
