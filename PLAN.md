# PLAN

## 次の実装イテレーション
保存済みスレッドを PDF 上の該当箇所と結びつけ、一覧または PDF 上から再発見できる状態まで進める。

## 今やる範囲
- スレッド作成時に、`pageNumber`、`selectedText`、`contextBefore`、`contextAfter` を保存する。
- 保存済みスレッドに対応する位置マーカーを、対象ページの右端ピンとして PDF 上に表示する。
- 右ペインのスレッド一覧をクリックすると、対象ページへスクロールし、対応するピンとスレッドを強調する。
- PDF 上の位置マーカーをクリックすると、対応するスレッドを右ペインで強調する。
- 位置情報を持たない既存スレッドは一覧表示のみとし、再発見対象外として扱う。
- 既存の作成、保存、復元の流れを壊さずに追加する。

## 完了条件
- 新規スレッド作成時に、再発見に必要な位置情報を保存できる。
- 保存済みスレッドに対応する右端ピンが PDF 上に表示される。
- 右ペイン一覧から、対象ページへ移動し、対応するピンとスレッドを見つけ直せる。
- PDF 上のピンから、対応するスレッドを右ペインで見つけ直せる。
- 位置情報のない旧スレッドは再発見不可として区別表示できる。
